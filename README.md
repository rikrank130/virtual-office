# みんなのひろば（シニア交流バーチャルスペース プロトタイプ）

oVice / Gather のような2Dバーチャル空間を、シニアでも迷わず使えるよう徹底的に単純化した交流スペースの試作です。
要件は [REQUIREMENTS.md](./REQUIREMENTS.md) を参照。

## 動かし方

```bash
npm install
npm run dev:all # 必ず allで実行してください
```

ビルド: `npm run build` → `npm run preview`

## 試せること（プロトタイプ範囲）

- なまえと顔を選んで即入室（登録不要）
- 行きたい場所を**タッチ/クリック**するだけで歩く（キー操作なし）
- 他の人に**近づくと自動で会話パネル**が開く（離れると閉じる）
- 定型あいさつボタン＋自由入力。相手はダミーで自動返信
- 「お茶の間／囲碁・将棋／健康体操／写真じまん」の話題ゾーン

※ 通信なし・ダミーデータのみ。リアルタイム接続、音声/ビデオ、保存、認証は未実装（次フェーズ）。

## 構成

- `src/data.js` — 空間サイズ・ゾーン・ダミーの人々
- `src/App.jsx` — 移動アニメーション・距離判定・全体レイアウト
- `src/EntryScreen.jsx` — 入室画面
- `src/ChatPanel.jsx` — 会話パネル
- `src/styles.css` — 大きな文字・高コントラストのシニア向けUI
